+ 口哨

+ 5寸水管钳

+ 纸巾

+ 多功能指甲钳

#### 最外层中间隐藏包

+ AGM功能机

+ 充电器

+ 防水记事本

+ 细记号笔(一支)

#### 最外层

+ 一次性N95口罩(3个)

+ 便宜单筒望远镜

+ 便宜战术笔

+ 手电(OLight i3T, 使用7号干电池供电)

+ 7号干电池(4节，防水密封袋包装)

+ 折叠手术刀(未装刀片)

+ 茶苯海明片(晕车药)

+ 电工胶带

+ 100元

+ 一截伞绳
